# LinkedIn Dashboard v3

Bản dashboard tối giản, tập trung vào dữ liệu đã scan.

## Thay file

Copy đè 3 file sau vào thư mục `frontend/`:

- `index.html`
- `styles.css`
- `app.js`

Giữ nguyên `config.js` hiện tại.
